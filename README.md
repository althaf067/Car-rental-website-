🚗 DriveEase — Car Rental System

DriveEase is a modern Car Rental Management System built using HTML, CSS, and JavaScript.
It provides an intuitive dashboard to manage cars, bookings, customers, and business operations in one place.

---

📌 Features

- 📊 Dashboard Overview
  
  - Total cars, available cars, rented cars
  - Revenue tracking
  - Recent bookings & activity timeline

- 🚘 Fleet Management
  
  - Add, remove, and manage cars
  - Filter by availability (Available / Rented / Maintenance)
  - Search functionality

- 📅 Booking System
  
  - Create new bookings
  - Automatic price calculation
  - Booking status tracking (Active, Completed, Cancelled)

- 👤 Customer Management
  
  - Add and manage customers
  - Track booking history per customer

- ⚙️ Admin Settings
  
  - Customize app name, currency, admin name
  - Reset all data with demo reload

- 🔔 UI Enhancements
  
  - Toast notifications
  - Responsive design
  - Clean modern UI

---

🛠️ Tech Stack

- Frontend: HTML5, CSS3, JavaScript (Vanilla)
- Styling: Custom CSS with modern UI design
- Data Handling: In-memory JavaScript state (no backend)
